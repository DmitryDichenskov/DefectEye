# Система обнаружения брака деталей на производстве

**Практическая часть дипломной работы**

## Описание проекта

Разработана автоматическая система обнаружения дефектов на металлических поверхностях с использованием сверточных нейронных сетей YOLOv11. Система включает веб-интерфейс для анализа изображений и API для обработки видеофайлов.

## Основные возможности

- Интерактивный веб-интерфейс для анализа изображений
- API для пакетной обработки видеофайлов
- Автоматическое сохранение результатов детекции
- Защита API паролем
- Поддержка форматов MP4, AVI, MOV, MKV, WebM

## Используемые технологии

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Детекция объектов | YOLOv11 (Ultralytics) | Обнаружение дефектов |
| Веб-интерфейс | Gradio | Интерактивный UI |
| REST API | Flask | Обработка видео |
| Язык | Python 3.8+ | Основная разработка |

## Установка и запуск

### 1. Установка зависимостей

```bash
pip install gradio ultralytics flask opencv-python
```

### 2. Структура проекта
your_project/
├── best.pt # Обученная модель YOLO
├── app.py # Основной скрипт
└── /path/to/video/ # Папка с видео файлами


### 3. Запуск системы

```bash
python app.py
```

### 4. Доступ к интерфейсам

- **Gradio UI**: http://localhost:7860
- **Flask API**: http://localhost:5000/analyze_video

## Использование

### Анализ изображений (Gradio UI)

1. Откройте http://localhost:7860
2. Загрузите изображение металлической поверхности
3. Получите результат с разметкой дефектов
4. Результат сохраняется в папку `web_results/`

### Анализ видео (Flask API)

```bash
curl -X POST http://localhost:5000/analyze_video \
  -H "Content-Type: application/json" \
  -d '{"password": "your_secret_password"}'
```

**Пример ответа:**
```json
{
  "status": "started", 
  "videos": 3
}
```

Результаты сохраняются в `web_results/video_results/`.

## Конфигурация

| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| `MODEL_PATH` | Путь к модели YOLO | `best.pt` |
| `VIDEO_DIR` | Папка с видео | `/path/to/your/video/folder` |
| `SAVE_DIR` | Папка результатов | `web_results/` |
| `PASSWORD` | Пароль API | `your_secret_password` |
